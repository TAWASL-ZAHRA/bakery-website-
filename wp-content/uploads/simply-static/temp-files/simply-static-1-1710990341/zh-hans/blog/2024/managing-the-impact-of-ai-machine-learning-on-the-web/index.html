分析人工智能与机器学习对 Web 的影响

Part of Web Architecture
切换语言：
  • English

作者及发布日期

由:
  • Dominique Hazaël-Massieux
发布:
2024年3月13日

过去几个月里，我们看到了大量与人工智能（大多基于机器学习模型的）系统相关的消息。

这些系统与作为平台的 Web 紧密结合：许多模型是从大规模抓取的 Web 内容中训练出来的，通过 Web 接口分发或显示，很多情况下用于生成迅速在 Web 上发布的内容。

这些交叉对 Web 产生了诸多系统性影响，引发了一系列关于它们如何改变我们所知的 Web 的重要讨论，其中有利有弊。

这些讨论分散在多个社区，因此有时很难全面了解问题空间。为了推动这些讨论更快走向具体成果，我写下了我从这些围绕人工智能和 Web 的讨论中所发现的系统性影响，以及一些早期的管理建议，这些建议涉及到标准化工作的开展和协调。

作为这些对话的切入点，W3C 团队今天发布了一份文档，名为《AI 与 Web：理解和管理机器学习模型对 Web 的影响》。我们诚邀全球社区就此提供反馈，帮助我们改善这项分析。 欢迎于2024年6月30日前反馈意见与建议。

我们希望提供一个广泛的交流平台，协助社区形成一条可靠路线，巩固 Web 在迅速发展的信息生态中的基础地位。
