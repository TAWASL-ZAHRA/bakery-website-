招聘：无障碍内容专家（全球）、Web 技术专家（美国）

切换语言：
  • English

作者及发布日期

发布:
2024年3月19日

W3C 现公开招聘以下两个职位，加入高度协作的全球 Web 无障碍标准化团队（WAI）：

  • 无障碍内容专家：致力于数字无障碍工作、全职岗位、可全球全程工作。
  • Web 技术专家：负责无障碍标准的实现工作（通过 GitHub）、项目周期6-18个月、全职岗位、可在美国远程工作。
