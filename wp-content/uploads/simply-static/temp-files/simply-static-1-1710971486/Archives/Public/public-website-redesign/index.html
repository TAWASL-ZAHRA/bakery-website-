public-website-redesign@w3.org Mail Archives

To send feedback on W3C's Website redesign, please submit each comment as issues. If you are not able to use GitHub, please email public-website-redesign@w3.org.

Everyone is welcome to send feedback, and you can read the feedback others have sent by browsing these archives.

If you wish to send feedback privately, please write to team-website-redesign@w3.org.
